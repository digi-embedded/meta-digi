Digi Embedded Yocto provides the following image recipes:

  * dey-image-qt: graphical QT image

    By default the image is XWayland-based so it provides a full Weston
    desktop environment.

  * dey-image-lvgl: graphical LVGL image

    By default the image is XWayland-based so it provides a full Weston
    desktop environment.
